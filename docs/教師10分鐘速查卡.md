# 教師 10 分鐘速查卡｜NPDL 評量設計工作室

正式站：[https://evan645test.github.io/npdl-assessment-studio/](https://evan645test.github.io/npdl-assessment-studio/)  
完整說明：[使用說明書.md](./使用說明書.md)

---

## 0–1 分｜開站與 AI

1. 開啟網站（預設＝**課程設計**）
2. 右上角齒輪 → 選模型 → 貼 API Key → 完成設定
3. 首次呼叫 AI：看傳送內容 → **同意並繼續**

> 金鑰只在本機瀏覽器；公用電腦用完請清除。

---

## 1–7 分｜課程設計（跟上方主按鈕走）

| 分 | 做什麼 | 按哪裡 |
|----|--------|--------|
| **1** | 填年級／學科／單元／主題／**3–5 關鍵字** | 上方：**開始分析課程關鍵字** |
| **2** | 初次校準 108＋6Cs | 上方：**進行 108 課綱與 6Cs 校準** |
| **3** | 各選 2 項表現／內容＋選 6Cs；若改選 | 選取後：**依教師調整重新校準** |
| **4** | 生成並鎖定學習終點 | 上方生成 → 底部：**確認學習終點…** |
| **5** | 建立並鎖定評量證據 | 上方建立 → 底部：**確認證據…** |
| **6** | 產生課前敘述語＋Q1–Q4 | 上方：**AI 產生課程敘述與完整課前評量** |
| **7** | 填節數／分鐘／限制 → 藍圖 | **欄位下方**：**AI 產生單元節次藍圖** → **確認藍圖…** |

**點擊口訣**

| 類型 | 位置 |
|------|------|
| 純產生 | 上方主按鈕（區內多半只寫「請用上方…」） |
| 先填後產 | 欄位填完，按鈕在**欄位下方** |
| 審閱確認 | 內容看完，確認鈕在**後面** |

**可選（有時間）**：交付中心 → 複製 Canvas 提示 → 開 Gemini 貼上產生教案／學習單（App 內不編教案）。

---

## 7–9 分｜帶入＋課後

1. **帶入評量設計** → 確認帶入  
2. （選填）實際教學與原設計不同之處  
3. **分析課程與前測，產生課後評量**

| 前後測 | 在哪產生 |
|--------|----------|
| **課前** | 課程設計 → 課前題組 |
| **課後** | 評量設計 → 產生課後評量 |

Q1–Q4 階梯：**概念理解 → 行動應用 → 生活遷移**

---

## 9–10 分｜帶走成果

擇一即可：

- 下載／列印 **Markdown 或 PDF**
- **Google Forms**：建立課前／課後問卷（失敗用「重試未完成問卷」）
- （選）收藏題庫、複製題目 Markdown

---

## 紅線（必守）

- 不存／不上傳**學生姓名**
- 改選課綱後要**重新校準**，下游才會更新
- 改終點／證據／限制後，藍圖與 Canvas 可能過期 → **重產並再確認**
- 單純切換「課程／評量」分頁**不會**覆蓋資料；只有「帶入」才會更新評量脈絡

---

## 卡住時

| 狀況 | 先檢查 |
|------|--------|
| AI 失敗 | 金鑰、模型、連線測試 |
| 進不了證據 | 是否已確認學習終點 |
| 進不了課前 | 是否已確認證據 |
| Canvas 停用 | 是否已確認藍圖；上游是否過期 |
| 畫面像舊的 | 強制重新整理 |
